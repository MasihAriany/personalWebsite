Mobile Portfolio

Udacity's Front-End Nanodegree August 2017

challenge, was to optimize a given portfolio page for speed!
In particular, optimizing the critical rendering path and making the page render as quickly as possible by applying the techniques I've picked up in the Critical Rendering Path course of Udacity's Nanodegree Program.

Online Demo
Github Repo

Purpose of the project

optimize a provided website with a number of optimization- and performance-related issues so that it achieves a target PageSpeed score and runs at 60 frames per second and a score of above 90 from 100 in Google's pageSpeed.

Implemented Optimizations

  • The Critical Rendering Path Optimized
  • Rendering Blocking CSS Avoided (For all except the main CSS file)
  • JavaScript Optimized
  • The size of text Reduced
  • Images Optimized
  • used HTTP Caching
