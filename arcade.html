Classical Arcade Game

Udacity's Front-End Nanodegree August 2017

Clone of platform arcade game Frogger. Coded in JavaScript using the pseudo-classical pattern.
Game was developed inside an HTML5 canvas utilizing the best practices of Object-oriented JavaScript.

Online Demo
Github Repo

Project features

  • Move the player across the screen with the arrow keys.
  • Player has four lives, showed by heart icons right below the page
  • Colliding with moving bugs take one life of the player and reset the position of the player
  • Enemies(Bugs) move in different speed in lines.
  • By loosing the last life, game will pause for a moment and reset to initial state.
