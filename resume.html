Interactive Resume

Udacity's Front-End Nanodegree August 2017

Resume data in four javaScript objects are stored.
The whole page is generated dynamically by JavaScript and JQuery according to the stored data.

Online Demo
Github Repo

Purpose of the project

An interactive resume application that reads my resume content from a JSON file and dynamically displays that content within a provided template.
It was implemented in a way to use objects, functions, conditionals, and control structures to compose the content that will display on the resume.
